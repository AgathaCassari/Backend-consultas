# Backend Consultas

API REST desenvolvida com Spring Boot.

## Funcionalidades
- CRUD de Pacientes
- CRUD de Especialidades

## Tecnologias
- Java
- Spring Boot
- JPA / Hibernate
- H2 Database

## Como rodar
- Rodar a classe BackendConsultasApplication
- Acessar: http://localhost:8080

## Endpoints
/pacientes
/especialidades
